# Speech-Recognition
This is a project created using SpeechRecognition with the intention of showing how this simple engine library works

### Funcionalities:
  - The algorithm listens to the microphone and responds according to the pattern words created

### Installation:
  - Download the correct PyAudio for your system and Python version from the [library](https://pypi.org/project/PyAudio/#files)

  - Install SpeechRecognition: ```pip install SpeechRecognition```
  - Install PyAudio, example: ```pip install C:\Downloads\PyAudiofile.whl```
